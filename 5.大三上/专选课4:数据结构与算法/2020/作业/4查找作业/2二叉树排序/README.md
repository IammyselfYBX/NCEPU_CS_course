# 编写二叉排序树相关算法：
- 1.画出一个二叉排序树，写出对应的先序字符串
- 2.输入先序字符串，编写递归算法创建二叉树
- 3.编写算法，输出二叉排序树的中序遍历结果
- 4.编写算法，在二叉排序树上查找给定值，找到返回该结点指针，找不到返回查找路径的最后一个结点指针。
